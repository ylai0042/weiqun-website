# 广西伟群帐篷制造有限公司网站

## 项目简介

这是一个为广西伟群帐篷制造有限公司设计的现代化、多语言支持的官方网站。网站采用简洁、美观、有条理的设计理念，提供完整的公司介绍、产品展示和客户服务功能。

## 功能特性

### 🌐 多语言支持
- **中文（简体）** - 主要语言
- **English** - 英文支持
- **日本語** - 日文支持  
- **한국어** - 韩文支持
- 自动语言检测和记忆用户偏好
- 平滑语言切换动画

### 🎨 现代化设计
- 响应式设计，支持所有设备
- 简洁优雅的视觉风格
- 专业的配色方案
- 流畅的动画效果
- 优秀的用户体验

### 📱 完整功能
- **公司介绍** - 企业历史、使命愿景、发展历程
- **产品中心** - 分类展示各类帐篷产品
- **服务范围** - 设计、制造、质量控制、配送全流程
- **联系我们** - 多种联系方式，在线表单
- **PDF下载** - 产品资料和公司介绍下载

### 🛠 技术特点
- 纯HTML5 + CSS3 + JavaScript
- 无依赖框架，轻量级
- SEO友好的结构
- 快速加载速度
- 跨浏览器兼容

## 使用方法

### 1. 本地运行
直接在浏览器中打开 `index.html` 文件即可预览网站。

### 2. 服务器部署
将所有文件上传到Web服务器，确保文件结构完整。

### 3. 自定义配置
- 修改 `scripts/multilang.js` 中的翻译内容
- 更新 `scripts/products.js` 中的产品信息
- 调整 `styles/main.css` 中的样式和配色

## 维护说明

### 添加新产品
1. 在 `scripts/products.js` 的 `products` 数组中添加新产品对象
2. 包含所有语言的名称、描述和规格信息
3. 指定正确的产品分类

### 修改翻译内容
1. 编辑 `scripts/multilang.js` 中的 `translations` 对象
2. 确保所有语言的键值对应
3. 测试语言切换功能

### 更新样式
1. 主要样式在 `styles/main.css` 中
2. 多语言特定样式在 `styles/multilang.css` 中
3. 遵循现有的CSS变量和命名规范
