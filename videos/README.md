# 视频文件说明

## 视频文件位置

请将您的Hero区域视频文件放在此文件夹中。

## 推荐视频文件

- **文件名**: `hero-video.mp4` (主要格式)
- **备用格式**: `hero-video.webm` (可选，用于更好的浏览器兼容性)

## 视频要求

### 技术规格
- **格式**: MP4 (H.264编码) 或 WebM
- **分辨率**: 1920x1080 (Full HD) 或更高
- **宽高比**: 16:9 (推荐)
- **时长**: 15-60秒 (循环播放)
- **文件大小**: 建议控制在 5-10MB 以内（可通过压缩优化）

### 内容建议
- 展示公司产品（帐篷、快幕秀等）的实际使用场景
- 户外活动、展览、活动的场景
- 产品搭建过程
- 客户使用产品的场景

### 优化建议
1. 使用视频压缩工具（如 HandBrake）减小文件大小
2. 确保视频在移动设备上也能流畅播放
3. 视频内容应该清晰展示产品特点
4. 避免过于复杂的画面，保持简洁专业

## 备用图片

如果视频无法加载，系统会自动使用备用图片：
- **位置**: `images/hero-backup.jpg`
- **规格**: 1920x1080 或更高分辨率

## 使用说明

1. 将视频文件命名为 `hero-video.mp4` 并放在此文件夹
2. 可选：添加 `hero-video.webm` 格式以提高兼容性
3. 刷新网站即可看到视频背景效果

## 注意事项

- 确保视频文件有适当的权限
- 视频会自动循环播放
- 视频在移动设备上可能会降级为静态背景以节省流量
- 建议定期更新视频内容以保持网站新鲜感
