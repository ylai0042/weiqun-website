## 🎯 使用建议

### 1. 筛选目标客户
根据以下标准筛选：
- **产品匹配度：** 选择与您产品类型匹配的公司
- **业务类型：** 优先联系进口商和分销商
- **地理位置：** 考虑物流和时区因素
- **公司规模：** 根据您的产能选择合适的客户规模

### 2. 联系前准备
- ✅ 访问目标公司官网，了解其业务和产品线
- ✅ 准备详细的产品目录（PDF格式）
- ✅ 准备公司介绍资料
- ✅ 准备样品或产品图片
- ✅ 了解目标公司的采购习惯和需求

### 3. 联系方式
1. **Email联系：** 使用提供的联系模板
2. **官网联系表单：** 通过公司官网的联系表单
3. **电话联系：** 如有公开电话，可先电话沟通
4. **LinkedIn：** 通过LinkedIn建立专业联系

### 4. 跟进策略
- 首次联系后1周内如无回复，发送礼貌的跟进邮件
- 保持专业和耐心的态度
- 根据客户反馈调整产品介绍和报价策略

## 📧 联系模板使用指南

### 使用步骤：
1. 打开 `联系模板_Contact_Template.md`
2. 复制适合的模板（英文或中文）
3. 替换模板中的占位符：
   - `[您的姓名]` → 您的真实姓名
   - `[您的职位]` → 您的职位
   - `[买家公司名称]` → 目标公司名称
   - `[买家姓名]` → 目标联系人姓名（如已知）
   - `[具体行业/市场细分]` → 目标公司所在行业

4. 添加产品目录作为附件
5. 发送前仔细检查拼写和语法

### 模板特点：
- ✅ 专业、礼貌、简洁
- ✅ 突出公司优势和产品特点
- ✅ 不含敏感个人信息
- ✅ 符合国际商务礼仪
